# v-safe ai demo

This contains everything you need to run the V-safe AI demo locally.

View your app in AI Studio: https://ai.studio/apps/ad0417b9-aed8-46c4-b565-7ea122c83fbf

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
